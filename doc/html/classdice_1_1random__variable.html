Detailed Description

template<typename ValueType, typename ProbabilityType>
class dice::random_variable< ValueType, ProbabilityType >

Discrete random variable.

Template Parameters
ValueTypetype of a value of this variable. Requirements:
  • std::numeric_limits specialization
  • std::hash specialization
  • std::max, std::min specialization
  • convertible to ProbabilityType
ProbabilityTypetype of probability (rational number in [0, 1])

The ValueType is used as a key in a hash table. ProbabilityType is a value in this hash table. Probabilities of values sum up to 1.

This type is immutable. All public methods and functions which modify a variable return a new random variable.

Create a random variable Z = XdY.

X times roll a Y sided dice. Assumes X, Y are independent random variables.

This funciton implements a simple dynamic programming algorithm. The probability that we roll k with X throws of a Y sided die is the probability that we roll k - n with X - 1 throws of a Y sided die and then roll n on the last die for n = 1 to Y.

Parameters
num_dicenumber of dice X (independent of Y) Each value has to be a positive integer.
num_facesnumber of faces of each die Y (independent of X) Each value has to be a positive integer.
Returns
distribution of XdY
Exceptions
std::invalid_argumentif X or Y contain a 0 value.
